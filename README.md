# Mastra MCP Project

Proyek ini menggunakan Mastra MCP untuk pengembangan sistem terintegrasi.

## Teknologi yang Digunakan

- Mastra Core v0.6.4
- Mastra MCP v0.3.5
- TypeScript
- Node.js

## Dependensi Utama

- @mastra/core: ^0.6.4
- @mastra/mcp: ^0.3.5
- @mastra/memory: ^0.2.4
- @mastra/pg: ^0.2.4
- @ai-sdk/anthropic: ^1.2.1
- @ai-sdk/openai: ^1.3.9
- @openrouter/ai-sdk-provider: ^0.4.5

## Cara Memulai

1. Clone repository
```bash
git clone https://github.com/itlrt/mastra-mcp-project.git
cd mastra-mcp-project
```

2. Install dependensi
```bash
npm install
```

3. Jalankan dalam mode development
```bash
npm run dev
```

## Scripts

- `npm run dev` - Menjalankan aplikasi dalam mode development
- `npm test` - Menjalankan test suite
